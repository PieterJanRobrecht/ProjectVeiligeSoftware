Opstarten
---------
1. Alle verschillende projecten in eclipse importen
2. Kaart builden
3. TimeServiceServer in TimeServer opstarten
3. ServiceProviderMain in ServiceProvider.main opstarten
4. Client in Middleware opstarten

Uittesten
---------
1. In de GUI van de ServiceProvider een SP kiezen en op lock klikken
2. In de GUI van de Client op login klikken
3. Nadat in de GUI aangegeven wordt dat stap 3 gedaan is, verschillende attributen selecten en op submit klikken
